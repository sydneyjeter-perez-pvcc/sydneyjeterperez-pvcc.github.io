Chapter 4: Networking

Sydney Jeter-Perez

Item Definition
Network Classifications
  • LAN: Consists of a collection of computers in a single building or building complex.
  • MAN: A network of intermediate size (like one spanning a local community).
  • WAN: Links machines over a greater distance, like neighboring cities or all over the world.
Network Models
  • Bus network: Machines are all connected to a common communication line called a bus.
  • Star network: One machine serves as a central focal point to which all the others are connected.
Protocols
  • CSMA/CD Ethernet: This protocol dictates that each message be broadcast to all the machines on the bus.
  • CSMA/CA Wifi: A protocol to avoid transmission collisions and not eliminate them completely.
Combining networks
  • Repeater: Passes signals back and forth between the two original buses.
  • Bridge: A complicated repeater that looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side.
  • Switch: a bridge with multiple connections, allowing it to connect to several buses instead of two.
  • Router: Special purpose computers used for forwarding messages between networks while allowing each network to maintain its unique internal characteristics.
  • Gateway: The "point" at which one network is linked to an internet, serving as a passageway between the network and the outside world.
Client/server model
  • P2P (peer-to-peer): Involves processes that provide service to and receive service from the client.
  • Distributed system: Software units that execute as processes on different computers.
  • Cloud computing: The practice of using a network of remote servers hosted on the internet to store, manage, and process data.
Internet Addressing
  • Tier 1: Consists of a very high-speed, high-capacity, international WANS (the backbone of the Internet).
  • Tier 2: Connected to tier-1 ISPs and tend to be more regional in scope and less potent in their capabilities.
  • Tier 3: An access ISP that is an independent Internet (aka intranet) operated by a single authority that is in the business of supplying Internet access to individual homes and businesses.
IP Address
  • ICANN: A nonprofit corporation established to coordinate the Internet's operation.
  • DNS: A collection of name servers that are directories that provide address translation services to clients.
  • TLD (Top-level domain): The part of a domain name that comes after the last dot (ex. ".com" in "google.com")
  • Country-code TLDs: A two-letter TLDs for each specific country.
Internet protocols
  • FTP: An application for listing and copying files across the network.
  • HTTP: A set of rules that govern how web browsers and web servers exchange information over the Internet.
  • SMTP: Defines a way that two computers on the network may interact when transmitting an email message from one host to another.
World wide web
  • URL: A unique address given to a document in order to locate and retrieve them.
  • Client-side activities: Processes that occur on the user's device to allow them to fill out orders for example.
  • Server-side activities: Everything that happens on the server, instead of on the client.
Internet four-layer software
  • Application layer: Part of a system that allows users to work with data through programs and applications.
  • Transport layer: Ensures data packets are sent accurately and reliably between a sender and receiver.
  • Network layer: Part of the internet communications process that enables network-to-network connections.
  • Link layer: Lowest layer of the internet protocol suite that is responsible for the communication protocols and methods that are specific to a host's physical link.
Internet is...
  • Packet-switching network: A method of transmitting data over a network by breaking it into smaller packets, or messages, that are then sent individually.
  • Fault tolerant: A system can continue to operate even when one or more of its components fail.
  • Redundant: A system design in which a component is duplicated so if it fails, there will be a backup.
Cybersecurity
  • Malware: Malicious software.
  • Virus: Software that infects a computer by inserting itself into programs that already reside in the machine.
  • Worm: An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
  • Trojan horse: A program that enters a computer system disguised as a desirable program that is willingly imported by the victim.
  • Spyware: Software that collects information about activities at the computer it resides in and reports that information back to the instigator of the attack (also called sniffing software).
  • Phishing: A technique of obtaining information explicitly by simply asking for it.
  • DDoS attack: When the work of generating the attacks is distributed over many hosts.
Protection
  • Firewall: Filters traffic passage through a point in the network.
  • Spam filter: Firewalls to block unwanted email.
  • Proxy server: A software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
  • Antivirus software: Used to detect and remove the presence of known viruses and other infections.
  • Cryptography: The study of sending and receiving messages securely in the presence of adversaries.
  • Public key encryption: A method of encrypting and decrypting data using two keys - a public and private key.
